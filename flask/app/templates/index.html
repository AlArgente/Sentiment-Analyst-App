¡Bienvenido!

Esta aplicación te permitirá resolver problemas de análisis de sentimientos, además de probar modelos ya generados, o bien por usted, o bien por otros usuarios. Como usuario podrás subir ficheros, en formato csv, para así entrenar un modelo, siendo el usuario el que elija qué preprocesamiento desea aplicar, además de qué algoritmo quiere utilizar.

Un ejemplo de flujo de trabajo es el siguiente, en el que se muestra cual sería el flujo de trabajo para un nuevo usuario:



Las distintas funciones de preprocesamiento que te permite aplicar son:

  • To_Lower
  • DeleteAccent
  • Tokenize
  • Stemmer
  • TF-IDF
Además el usuario podrá elegir entre dos algoritmos de clasificación:
  • Bayes:
  • SVM:
Además se deberá indicar el idioma del archivo que ha sido cargado para que las funciones anteriores puedan funcionar correctamente. El usuario podrá elegir qué tipo de división se hará entre train-test de entre las disponibles, además de indicar qué nombre tendrá el modelo que se va a entrenar.

Para cada modelo modelo almacenado en la aplicación el usuario podrá ver tanto las funciones de preprocesamiento aplicadas como el algoritmo de entrenamiento utilizado. Además podrá subir nuevos datos para probar el modelo con un nuevo conjunto de datos, obteniendo después los resultados, recall, precision, f1_score, obtenidos para el conjunto de datos.
